API Documentation

GET /movies

Returns all movies.

GET /movies/{id}

Returns a movie by its ID.

GET /movies/{id}/stars

Returns all stars of a movie

GET /movies/ratings/{year}

Retrieves a list of movies ordered by rating in descending order, for a given year

GET /people

Returns a list of people

GET /people/{id}

Returns a person by their ID.

GET /people/{id}/movies

Returns all movies of a person
